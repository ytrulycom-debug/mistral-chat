# Mistral Chat — Interface locale IA

Interface de chat connectée à l'API Mistral AI, tournant en local avec Laragon/PHP.

## Structure du projet

- `index.php` — fichier principal (interface + appel API)

## Fonctionnalités

- Chat en temps réel avec Mistral AI
- Historique de conversation complet
- Gestion des erreurs API
- Interface responsive

## Stack technique

- PHP 8.3
- Apache (Laragon)
- API Mistral AI (mistral-small-latest)
- JavaScript vanilla

## Pages à créer

- `index.php` — page principale du chat
- `config.php` — configuration et clé API
- `history.php` — historique des conversations sauvegardées
- `export.php` — export des conversations en PDF ou Word
